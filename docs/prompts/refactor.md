# Prompt — Refatoração Localizada no Prumo

Você está trabalhando no projeto **Prumo**, um app de lançamentos financeiros por voz para pequenos empreendedores.

## Contexto do produto
- O fluxo principal do app é: voz → processamento → revisão → confirmação.
- O foco do produto é simplicidade extrema.
- O sistema ainda está em fase de refinamento do MVP.
- A prioridade é manter o fluxo principal confiável e fácil de usar.

## Objetivo da refatoração
Quero refatorar uma parte específica do código para melhorar:
- clareza;
- legibilidade;
- organização;
- manutenção futura.

## Regra principal
A refatoração **não deve mudar o comportamento do usuário**, a menos que isso tenha sido explicitamente pedido.
